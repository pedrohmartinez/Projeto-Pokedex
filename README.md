# 👾 Pokédex - IA Code Experience

Aplicação web interativa estilo Pokédex desenvolvida durante o workshop **IA Code Experience**. O projeto consome uma API externa de Pokémon e foi estruturado utilizando o conceito de **desenvolvimento agêntico**, onde agentes autônomos de Inteligência Artificial atuaram ativamente na co-criação, planejamento e estruturação do código.

---

## 🎯 Objetivo do Projeto

O principal objetivo foi explorar a sinergia entre o desenvolvimento de software tradicional e a engenharia de software orientada a agentes (Agentic AI). A aplicação realiza buscas dinâmicas em uma API pública para listar e exibir informações detalhadas sobre os Pokémons, validando a eficiência do desenvolvimento agêntico na aceleração do fluxo de trabalho e na qualidade do código produzido.

## 🚀 Tecnologias e Ferramentas

* **Desenvolvimento Agêntico:** IA baseada em agentes / Modelos Codex
* **Consumo de API:** PokéAPI
* **Frontend:** HTML5, CSS3, JavaScript

## 🧠 O que é Desenvolvimento Agêntico neste projeto?

Diferente do uso tradicional de LLMs como assistentes de chat comuns (onde você apenas faz perguntas isoladas), o desenvolvimento agêntico envolve o uso de agentes de IA capazes de planejar passos, executar tarefas, testar soluções e refatorar trechos de código de forma autônoma ou assistida. No workshop, essa abordagem foi utilizada para:

1. **Arquitetura:** Planejar e estruturar a fundação do projeto.
2. **Integração:** Implementar a lógica de consumo assíncrono da API externa.
3. **Refatoração:** Otimizar funções de busca e auxiliar na resolução de bugs (debugging).
